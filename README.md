# finance-tracker-mvp
1. Пітч та проблема
Більшість людей просто фіксують свої щоденні витрати, але не аналізують співвідношення між активами (те, що генерує дохід) та пасивами (те, що вимагає витрат), що заважає їм накопичувати капітал.
Наш вебзастосунок допомагає не лише вести облік транзакцій, а й чітко розділяти фінанси на активи та пасиви, формуючи мислення, орієнтоване на примноження багатства.
Цільова аудиторія - студенти, фрилансери та інвестори-початківці, які хочуть свідомо керувати своїми грошима.
