# Користувацький сценарій

* Користувач знаходиться на головній сторінці застосунку.

* 1. Натискає кнопку «Додати транзакцію».

  2. Вводить суму, обирає категорію (наприклад, «Актив: Депозит») і натискає «Зберегти».

* Новий запис з'являється в історії транзакцій, а діаграма загального капіталу автоматично оновлюється.
